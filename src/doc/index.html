数据接口开放平台

整合SQLServer、MySql、ORACLE三大关系型数据库，抹平数据后端差异。

快速上手 →

接口鉴权

基于JWT的接口鉴权，将用户身份状态与后端剥离，方便横向扩展，分布式部署。

快速高效

基于PHP7.0，根据不同使用场景精准提供细粒度的前后端接口数据缓存方案。

自动生成调用代码

一键生成前台调用代码,风格统一。函数名、变量、参数、接口注释自动适配。

满足各种应用场景

系统提供RPC/Restful调用方式，JSON/XML数据格式，HTTP/Websocket协议，新老项目平滑过渡。

内外网支持

目前已在公司多个信息化系统(Node后端、Web前端、微信小程序、微信网页开发)中应用，外网支持阿里云、新浪云等主流云平台。

通用化

与业务完全解耦，凝练多年前后端分离开发经验，满足各类数据报表、图表、数据表单操作等场景，数据输出格式统一，抹平接口格式差异。

// 接口列表：
// db.js
import axios from './axios';
/**
 *   @database: { 接口管理 }
 *   @desc:     { 测试数据-查询指定类型数据 }
 */
export const getSampleBar = data_type => axios({
  url: '/6/8d5b63370c.json',
  params: {
    data_type
  },
});


/** 数据量较大时建议使用post模式：
*
 *   @database: { 接口管理 }
 *   @desc:     { 测试数据-查询指定类型数据 }
 */
export const getSampleBar = data_type => axios({
  method: 'post',
  data: {
    data_type,
    id: 6,
    nonce: '8d5b63370c'
  },
});


/** NodeJS服务端调用：
*
 *   @database: { 接口管理 }
 *   @desc:     { 测试数据-查询指定类型数据 }
 */
module.exports.getSampleBar = data_type => axios({
  url: '/6/8d5b63370c.json',
  params: {
    data_type
  },
});
